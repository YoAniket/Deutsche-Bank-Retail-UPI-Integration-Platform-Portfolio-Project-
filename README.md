# Deutsche Bank Retail UPI Integration Platform (Portfolio Project)

Designed a native UPI payment platform for Deutsche Bank's retail banking application by documenting business requirements, business process flows (As-Is/To-Be), stakeholder analysis, and integration requirements for the bank's Core Banking System (CBS) with the UPI ecosystem.
Defined end-to-end payment journeys including customer onboarding, bank account linking, UPI PIN management, P2P/P2M transfers, QR payments, collect requests, transaction history, notifications, dispute resolution, and settlement workflows while incorporating security, compliance, and operational requirements aligned with Indian digital payment standards.

Deliverables: 
* BRD
* FRS
* Wireframes
* BPMN
* RTM
* API Integration
* UAT Scenarios
* Risk Register
* Use Cases
* User Stories
* Acceptance Criteria
* Data Dictionary
* Sequence Diagrams
* Activity Diagrams
* Entity Relationship Diagram
* Product Backlog
